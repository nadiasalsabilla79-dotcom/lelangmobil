6. TROUBLESHOOTING
================================================================================

MASALAH: Port 3000 sudah digunakan
SOLUSI:
  pnpm dev -- -p 3001
  (gunakan port lain)

MASALAH: Dependencies error saat install
SOLUSI:
  1. Hapus folder node_modules
  2. Hapus file pnpm-lock.yaml atau package-lock.json
  3. Install ulang: pnpm install

MASALAH: Tailwind classes tidak muncul
SOLUSI:
  1. Stop server (Ctrl+C)
  2. Jalankan ulang: pnpm dev

MASALAH: Images tidak muncul
SOLUSI:
  1. Pastikan file ada di folder /public
  2. Check path nya benar (case-sensitive)
  3. Refresh browser (Ctrl+F5)

MASALAH: Store tidak persist (data hilang setelah refresh)
SOLUSI:
  1. Check browser localStorage
  2. Clear cache browser
  3. Jangan gunakan incognito mode

MASALAH: Node.js tidak terdeteksi
SOLUSI:
  1. Install Node.js dari https://nodejs.org/
  2. Restart Command Prompt
  3. Coba lagi
