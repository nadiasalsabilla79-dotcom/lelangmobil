========================================
TROUBLESHOOTING
========================================

MASALAH: Build Failed di Vercel
SOLUSI: 
- Cek Build Logs di tab Deployments
- Pastikan semua environment variables sudah di-set
- Pastikan database Supabase sudah di-setup

MASALAH: Login redirect ke login terus
SOLUSI:
- Clear browser cookies
- Coba browser incognito/private
- Pastikan middleware.ts sudah ter-update

MASALAH: Database connection error
SOLUSI:
- Cek environment variables di Vercel
- Pastikan POSTGRES_PRISMA_URL benar
- Cek Supabase dashboard apakah database aktif
