TROUBLESHOOTING
========================================

‚ùå Error: "Internal server error"
‚úÖ Solusi: Database belum di-setup, jalankan SQL di Step 1

‚ùå Error: "Invalid credentials"
‚úÖ Solusi: 
   - Pastikan SQL sudah dijalankan
   - Gunakan password: password123
   - Email harus exact match

‚ùå Build Failed di Vercel
‚úÖ Solusi:
   - Pastikan ENV variables sudah di-set
   - Clear build cache dan redeploy
   - Cek build logs untuk detail error

‚ùå Login redirect ke login terus
‚úÖ Solusi:
   - Clear browser cookies
   - Coba incognito/private mode
   - Pastikan middleware sudah ter-update

‚ùå Database connection error
‚úÖ Solusi:
   - Cek POSTGRES_PRISMA_URL di Vercel
   - Pastikan Supabase database aktif
   - Test connection di Supabase dashboard
